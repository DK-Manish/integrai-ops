# IntegrAI Ops Documentation

Welcome to the documentation for **IntegrAI Ops**.

This documentation is maintained alongside the project and evolves as the application is built.

## Structure

| Folder | Purpose |
|---------|---------|
| `01-product-vision/` | Product vision, business problem, goals, features and roadmap |
| `02-architecture-portal/` | Interactive animated architecture website |
| `03-presentation/` | Recruiter presentation and showcase |
| `shared/` | Shared assets such as images, icons, styles and data |

## Objective

The goal is to document not only **what was built**, but also **why it was built**, **how it works**, and **the engineering decisions behind it**.

This documentation is intended for:

- Recruiters
- Hiring managers
- Technical interviewers
- Engineers exploring the repository
