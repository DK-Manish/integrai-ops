## Logo Usage

Always use the official IntegrAI Ops logo.

Maintain clear spacing around the logo.

Do not stretch or distort it.

Use the coloured version on dark backgrounds.

Use a monochrome version only when required.

---

## Design Style

- Dark theme
- Glassmorphism where appropriate
- Rounded corners
- Smooth animations
- Subtle gradients
- Enterprise dashboard aesthetic

---

## Documentation Theme

All project documentation, presentations and architecture diagrams should follow these branding guidelines.
